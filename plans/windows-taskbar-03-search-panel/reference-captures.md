# Windows UI 작업 표시줄 Search 패널 기준 캡처

> 이 문서는 `TaskbarSearchPanel`과 그 내부 secondary surface를 고정하는 기준 이미지 목록이다.

## 기준 캡처 세트

- `search-panel-default.png`
  - Search 패널 기본 상태
- `search-panel-query-results.png`
  - 검색어 입력 후 비어 있지 않은 결과 목록 상태
- `search-panel-query-detail.png`
  - 결과 행의 `>` 버튼으로 연 detail pane 상태
- `search-panel-query-detail-pinned.png`
  - detail pane에서 pin/unpin 라벨이 반전된 상태
- `search-panel-query-empty.png`
  - 검색어 입력 후 빈 결과 상태
- `search-result-context-menu.png`
  - 검색 결과 행 우클릭 메뉴 상태

## 비고

- `search-panel-query-detail.png`는 제목, source label, action list hierarchy가 읽혀야 한다.
- `search-panel-query-detail-pinned.png`는 pin/unpin 라벨 반전 기준으로 본다.
- `search-result-context-menu.png`는 root taskbar icon menu와 다른 종류의 compact menu다.
