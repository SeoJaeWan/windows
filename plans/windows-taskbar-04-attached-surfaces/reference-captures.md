# Windows UI 작업 표시줄 부착형 surface 기준 캡처

> 이 문서는 `TaskbarHoverPanel`과 `TaskbarIconContextMenu`를 고정하는 기준 이미지 목록이다.

## 기준 캡처 세트

- `taskbar-hover-preview.png`
  - 활성 아이콘 hover 미리보기 상태
- `taskbar-icon-context-menu.png`
  - 활성 아이콘 root context menu 상태

## 비고

- hover preview는 menu가 아니라 compact preview strip으로 읽혀야 한다.
- `taskbar-icon-context-menu.png`는 root icon menu 기준이다.
- recent item의 2차 우클릭 menu는 아직 안정적으로 재현하지 못해 이 문서에 포함하지 않는다.
