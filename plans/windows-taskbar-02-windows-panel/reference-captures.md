# Windows UI 작업 표시줄 Windows 패널 기준 캡처

> 이 문서는 `TaskbarWindowsPanel`과 그 내부 secondary surface를 고정하는 기준 이미지 목록이다.

## 기준 캡처 세트

- `start-panel-default.png`
  - Windows 패널 기본 상태
- `start-panel-all.png`
  - Windows 패널 `all` 상태
- `start-panel-all-index.png`
  - Windows 패널 `all` 상태의 알파벳 인덱스 버튼 군이 분명히 보이는 상태
- `start-panel-query-empty.png`
  - Windows 패널 검색어 입력 후 빈 결과 상태
- `start-panel-pinned-updated.png`
  - 검색 결과를 시작 화면에 고정한 뒤 pinned 영역이 바뀐 상태
- `start-panel-pinned-4items.png`
  - pinned tile이 4개 보이는 상태
- `start-all-item-context-menu.png`
  - `all` 목록 항목 우클릭 메뉴 상태
- `start-pinned-context-menu-right-only.png`
  - 첫 번째 pinned tile 우클릭 메뉴 상태
- `start-pinned-context-menu-left-right.png`
  - 두 번째 pinned tile 우클릭 메뉴 상태
- `start-pinned-context-menu-forward-left-right.png`
  - 세 번째 pinned tile 우클릭 메뉴 상태
- `start-pinned-context-menu-forward-left.png`
  - 네 번째 pinned tile 우클릭 메뉴 상태

## 비고

- `start-panel-all.png`는 목록 구조를, `start-panel-all-index.png`는 인덱스 버튼 군을 우선 본다.
- pinned tile 메뉴는 위치에 따라 move action subset이 달라진다는 점을 반드시 유지한다.
